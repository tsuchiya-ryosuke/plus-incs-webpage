# VisionWell+ ウェブサイト 画像要件ドキュメント

このドキュメントでは、株式会社PLUSのウェブサイトで使用する画像の種類、サイズ、内容を定義します。

## 1. トップページ（index.html）

### 1-1. ヒーローセクション背景画像（オプション）
- **用途**: トップページの第一印象を決める背景画像
- **推奨サイズ**: 1920×1080px（Full HD）
- **形式**: JPG または WebP
- **内容**:
  - 学校の教室や図書館で生徒がタブレット/PCを使っている様子
  - または、先生と生徒が面談している穏やかなシーン
  - 明るく、前向きな雰囲気
- **配置場所**: ヒーローセクション（75-100行目）
- **代替案**: 現在のグラデーション背景のままでも可

### 1-2. 主要機能のスクリーンショット（4枚）

  - 「今学期頑張ったこと」などのプロンプトが見える
  - オートセーブ表示がある
  - AIサマリーの例が見える
- **配置場所**: 主要機能セクション（291-318行目）

#### B. 面談チャット＆アジェンダ生成画面
- **用途**: チャット機能の説明
- **推奨サイズ**: 800×600px
- **形式**: PNG（透過背景）または JPG

  - アジェンダ生成ボタンが見える
- **配置場所**: 主要機能セクション（321-347行目）

#### C. 面談モード・調査票作成画面

#### D. 進路情報検索（RAG機能）画面

## 画像作成のガイドライン

### デザイン原則
- **色**: 既存のカラースキーム（青系: #1e3a8a, #3b82f6, #60a5fa）と調和
- **スタイル**: クリーン、モダン、プロフェッショナル
- **一貫性**: すべての画像で統一感を保つ
- **レスポンシブ**: モバイルでも見やすいシンプルなデザイン

### ファイル命名規則
```
hero-background.jpg
feature-semester-report.png
feature-chat-agenda.png
feature-interview-mode.png
feature-rag-search.png
scene-1-student-teacher.png
scene-2-reflection.png
scene-3-recommendation.png
scene-4-new-teacher.png
support-onboarding.svg
support-operation.svg
support-improvement.svg
document-thumbnail.png
```

### 画像最適化
- **ファイルサイズ**: 各画像 200KB 以下を目標
- **フォーマット**:
  - 写真・スクリーンショット: JPG または WebP
  - アイコン・イラスト: SVG または PNG
- **代替テキスト**: すべての画像に適切なalt属性を設定

## 画像の配置方法

### HTML例
```html
<!-- 主要機能セクション -->
<div class="bg-white rounded-xl p-8 shadow-sm hover:shadow-md transition">
    <div class="mb-6">
        <img src="images/feature-semester-report.png"
             alt="学期レポート入力画面のスクリーンショット"
             class="w-full rounded-lg shadow-md">
    </div>
    <h3 class="text-xl font-bold text-blue-900">学期レポート</h3>
    <!-- 既存の内容 -->
</div>
```

### CSSクラス
```css
.screenshot-shadow {
    box-shadow: 0 10px 40px rgba(0, 0, 0, 0.1);
    border-radius: 8px;
}

.screenshot-border {
    border: 1px solid #e5e7eb;
}
```

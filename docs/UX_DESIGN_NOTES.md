## 1. 想定ユーザー（ペルソナ）

### プライマリユーザー: 進路面談担当の先生 / 3年生学年主任

**属性**
- 年齢: 30〜50代
- 役職: 進路指導主事、学年主任、担任教諭
- ICTスキル: 中程度（Excel、メールは使える。新しいツールには慎重）

**日常の課題**
- 授業準備、部活指導、校務で多忙
- 進路面談の準備に時間がかかる（生徒の記録を探し回る）
- 面談時間が限られている中で、事務連絡だけで終わってしまう
- 生徒の本音や悩みに寄り添いきれないジレンマ
- 推薦文作成の負担が大きい
- 新任の先生は進路指導の経験が少なく不安

**心理状態**
- 「生徒一人ひとりと向き合いたいけど、時間がない」
- 「もっと効率的に進路指導ができないか」
- 「新しいツールを導入して失敗したくない」
- 「予算が限られている中で、費用対効果を説明できるか」

### セカンダリユーザー: 進路に悩む高校生

**属性**
- 年齢: 15〜18歳（高校1〜3年生）
- 学年による違い:
  - 1年生: 漠然と将来を考え始める時期
  - 2年生: 文理選択や進路の方向性を決める時期
  - 3年生: 具体的な志望校・就職先を決める時期
- デジタルリテラシー: 高い（スマホ・SNSネイティブ世代）

**日常の課題**
- 将来やりたいことが明確でない、または複数あって絞れない
- 進路について相談したいが、先生に面談時間を取ってもらうのが難しい
- 親の期待と自分のやりたいことのギャップに悩む
- 成績や偏差値だけで進路を決めていいのか不安
- 本音を先生に直接言いにくい（恥ずかしい、評価が気になる）
- 進路面談で毎回同じ質問をされる（「最近どう？」から始まる）

**心理状態**
- 「将来のことを考えなきゃいけないけど、何から始めればいいかわからない」
- 「本当は〇〇に興味があるけど、言い出しにくい」
- 「先生は忙しそうだから、あまり時間を取らせたくない」
- 「自分のことをちゃんとわかってくれているのかな」
- 「進路面談って、結局書類の確認で終わっちゃう」

## 1.5 生徒目線で見るVisionWell+のメリット

### AIチャットで得られる価値

**① 24時間いつでも相談できる安心感**
```
深夜に進路のことが急に不安になっても、
AIチャットなら気軽に相談できる。
「こんなこと聞いていいのかな」という遠慮も不要。
```

**② 本音を言いやすい環境**
```
先生に直接言いにくいことも、AIなら正直に話せる。
「実は親の希望と違う進路を考えている」
「成績は良くないけど、挑戦したい学校がある」
評価を気にせず、自分の気持ちを整理できる。
```

**③ 自分のペースで考えられる**
```
面談の限られた時間ではなく、
じっくり自分と向き合える。
考えがまとまってから先生に相談することも可能。
```

### 先生が生徒カルテを使うことで得られる価値

**① 「わかってくれている」という実感**
```
面談の最初から「最近どう？」ではなく、
「前回話してた〇〇の件、その後どうなった？」
→ 自分のことを覚えていてくれる安心感
```

**② 本質的な対話に時間を使える**
```
事務連絡や確認作業が減り、
「将来どうしたいか」「何に悩んでいるか」
本当に話したいことに面談時間を使える。
```

**③ 見えない悩みに気づいてもらえる**
```
AIチャットで話した内容を先生が把握。
自分から言い出せなかった悩みも、
先生の方から「〇〇について話そうか」と
声をかけてもらえる。
```

### 生徒にとっての「Before → After」

| 項目 | Before（従来） | After（VisionWell+導入後） |
|------|---------------|---------------------------|
| 相談のタイミング | 面談の予約が必要、先生の時間に合わせる | 24時間いつでもAIに相談可能 |
| 本音の伝えやすさ | 先生に直接言いにくい、評価が気になる | AIには正直に話せる。整理してから先生に相談 |
| 面談の内容 | 事務連絡や確認が多い、本題に入る前に時間切れ | 最初から本質的な進路相談ができる |
| 先生の理解度 | 毎回同じ質問から始まる、覚えてもらえていない感 | 自分のことを把握してくれている安心感 |
| 悩みへの対応 | 自分から言い出さないと気づいてもらえない | 見えない悩みにも先生が気づいてくれる |

### 学期レポートで得られる価値

**機能概要**
```
学期ごとに生徒が振り返りを記録。
「今学期頑張ったこと」などのプロンプトに沿って入力。
AIが内容を要約し、3年間の成長の軌跡を可視化。
```

**① 3年生の面接対策に活用できる**
```
1年生から積み重ねた学期レポートが、
面接での自己PRや志望動機のネタになる。

「高校生活で頑張ったことは？」
→ レポートを見返せば、具体的なエピソードがすぐ見つかる。

自分の成長ストーリーを語れるようになる。
```

**② 自分の成長を実感できる**
```
「1年生のとき数学が苦手だったけど、今は得意科目に」
「プレゼンが得意になった」
過去の自分と比較することで、成長を客観的に認識。
面接で自信を持って話せる。
```

**③ 担任が変わっても情報が引き継がれる**
```
毎年担任が変わることは珍しくない。
でも3年生の担任が調査書（推薦書）を書く必要がある。

学期レポートがあれば、
1年・2年のときの様子も把握した上で、
説得力のある調査書を作成できる。
→ 生徒にとっても、自分のことをちゃんと書いてもらえる安心感。
```

### 学期レポートの「Before → After」

| 項目 | Before（従来） | After（VisionWell+導入後） |
|------|---------------|---------------------------|
| 面接対策 | 「頑張ったこと」を思い出すのに苦労 | 3年分のレポートからエピソードを選べる |
| 自己理解 | 自分の成長に気づきにくい | 過去との比較で成長を実感 |
| 担任の引き継ぎ | 新担任は生徒のことをゼロから把握 | レポートで1年・2年の様子も把握済み |
| 調査書作成 | 3年生の担任が1・2年の情報を集める必要 | 学期レポートにまとまっていて書きやすい |
| 推薦文 | 先生が1時間以上かけて作成 | AIが下書き生成、30分で完成 |

## 2. ユーザージャーニー

### Phase 1: 認知・訪問
**きっかけ**
- 同僚や教育委員会からの紹介
- 教育系メディアでの記事
- 検索（「進路指導 効率化」「AI 進路支援」など）

**サイト訪問時の期待**
- 「自分の悩みを理解してくれているか」を確認したい
- 具体的にどう楽になるのかを知りたい
- 費用が気になる（学校の予算は限られている）
- 実際に使っている先生の声を聞きたい

### Phase 2: 理解・共感
**ヒーローセクションでの体験**
```
先生の「時間」を創り、生徒の「本音」を引き出す。
事務作業はAIに任せ、先生は「先生にしかできない指導」へ。
```
→ 「自分の悩みを理解してくれている」と感じる

**共感セクションでの体験**
```
生徒一人ひとりと向き合いたい。
でも、そのための「時間」と「情報」が足りない。
```
→ 「そうそう、まさにこれ」と共感

```
先生が悪いわけではありません。
時間がない環境の中で、先生方は最善を尽くされています。
```
→ 「責められていない、理解してくれている」と安心

### Phase 3: 解決策の理解
**Before/After図解での体験**
- 情報収集・事務連絡 80% → 20%
- 本質的な相談 20% → 80%
→ 「具体的にこう変わるのか」と納得

**機能紹介での体験**
- 「生徒カルテ」: 面談準備を「ゼロ」にする
- 「AIチャット」: 見えない悩みを可視化する
→ 「機能」ではなく「変化」として理解

### Phase 4: 信頼・検討
**導入事例・お客様の声での体験**
```
「面談の20分すべてを、本質的な対話に使えるようになりました」
```
→ 「実際に使っている先生が効果を実感している」と信頼

**料金での体験**
```
2026年3月末まで完全無料
先行導入校特別価格: 200円/月（通常400円）
```
→ 「リスクなく試せる」「ワンコイン以下なら検討できる」と安心

### Phase 5: アクション
**CTAでの体験**
- 「無料トライアルに申し込む」
- 「資料ダウンロード」
→ 明確な次のステップを提示

## 3. UXデザイン原則

### 3.1 共感から入る
**悪い例**
```
VisionWell+は最新のAI技術を活用した進路支援システムです。
```

**良い例**
```
生徒一人ひとりと向き合いたい。
でも、そのための「時間」と「情報」が足りない。
```

→ ユーザーの課題を言語化し、「理解されている」と感じさせる

### 3.2 機能より変化を語る
**悪い例**
```
AIチャット機能が使えます
生徒カルテ機能があります
```

**良い例**
```
AIチャットがあるおかげで、先生は生徒の隠れた悩みを知ることができます
生徒カルテで事前に状況を把握。面談1分で本題に入れます
```

→ 常に「先生へのメリット」を主語にする

### 3.3 視覚で直感的に伝える
**Before/After時間配分図解**
- プログレスバーで時間配分の変化を可視化
- 80%→20%の逆転を一目で理解できる

**生徒カルテ・AIチャット画面イメージ**
- 実際の画面に近いモックアップを表示
- 「こう使うのか」と具体的にイメージできる

### 3.4 ハードルを下げる
**無料トライアル**
```
2026年3月末まで完全無料
無料期間中の解約も自由
```
→ 「試してダメなら辞められる」という安心感

**価格の見せ方**
```
通常価格 400円 → 先行導入校特別価格 200円
ワンコイン以下で導入可能
```
→ 「高くない」と感じさせる

## 5. コピーライティングガイドライン

### トーン
- **共感的**: 上から目線ではなく、先生の立場に寄り添う
- **具体的**: 抽象的な表現を避け、数字や事例で語る
- **謙虚**: 「AIが全てを解決する」ではなく「AIがサポートする」

### 避けるべき表現
- ❌ 「革命的」「画期的」などの誇大表現
- ❌ 「先生方の負担を軽減」（上から目線）
- ❌ 「簡単に導入できます」（実際は学校ごとに課題がある）

### 推奨する表現
- ✅ 「先生の時間を創る」
- ✅ 「先生にしかできない指導に集中」
- ✅ 「まずはリスクゼロでお試し」

学校・先生が安心して使える
進路支援ツール

株式会社PLUSは、学校・先生が安心して使える進路支援ツールを提供しています。

生徒の思いや考えを日常的に記録し、進路指導主事・学年主任の先生が
「探す時間」ではなく「向き合う時間」に集中できる環境をつくります。

こんな課題を解決します

生徒の課題

  • ● 自分の考えを整理するのが難しい
  • ● 面談の準備に不安がある
  • ● 過去の振り返りが散らばっている

先生の課題

  • ● 面談準備に時間がかかる
  • ● 進路指導の情報収集が大変
  • ● 推薦文作成の負担が大きい

学校の課題

  • ● 進路指導の質のばらつき
  • ● 生徒の成長の可視化が難しい
  • ● 情報の属人化

VisionWell+

進路支援を変える、3つの視点

生徒にとって

  • ✓ AIに相談しながら考えを整理できる
  • ✓ 自分の気持ち・価値観を言葉にできる
  • ✓ 面談の場で「うまく話せない」を減らせる

教員にとって

  • ✓ 生徒の記録を一元管理
  • ✓ 調査票作成時に、過去の記録を探し回らなくていい
  • ✓ 面談前に生徒像を把握できる

学校にとって

  • ✓ 進路面談の正味時間を最大化
  • ✓ 指導の質が属人化しにくい
  • ✓ 記録が蓄積され、学年・年度を越えて活用できる

主要機能

VisionWell+は、進路支援に必要な機能を統合的に提供します

学期レポート

学期ごとに生徒が振り返りを記録。AIが要約し、成長の軌跡を可視化します。

  • • オートセーブで安心
  • • 過去のレポートと比較可能
  • • AIサマリーで要約

面談チャット＆アジェンダ生成

AIとの対話で考えを整理。チャット内容から面談アジェンダを自動生成します。

  • • AIが適切な質問を投げかける
  • • アジェンダを先生と共有
  • • 面談がスムーズに始まる

面談モード・調査票作成

生徒情報を一画面で確認。AIが推薦文の下書きを生成し、作成時間を大幅削減。

  • • 共有アジェンダ・レポート・進路情報を統合表示
  • • AI生成で推薦文作成時間を短縮
  • • 複数バージョン管理

進路情報検索（RAG機能）

自然な言葉で検索し、最新の進路情報にすぐアクセス。面談中も即座に調べられます。

  • • 自然言語で検索可能
  • • 関連情報も表示
  • • 面談中にその場で情報提供
機能の詳細を見る

こんな場面で活用できます

1

高校1年生の初めての面談準備

生徒：ユキさん、担任：田中先生

ユキさんはチャットで「将来のことを考えたいけど、まだ何も決まってない」と入力。AIとの対話を通じて「動物が好き」「生物の授業が面白い」という気づきを得て、アジェンダを先生と共有。田中先生は事前に生徒の興味を把握し、面談で深い対話ができました。

2

学期ごとの振り返りで成長を実感

生徒：タケシさん（高校2年生）

タケシさんは学期末に学期レポートを記入。「今学期頑張ったこと」を書きながら「自分はプレゼンが得意になった」と気づきます。1年次のレポートを見返すと「数学が苦手だったけど、今は得意科目に」という成長を実感できました。

3

推薦入試の準備を効率化

生徒：アヤさん、進路指導：佐藤先生

佐藤先生はアヤさんの推薦文作成にAI生成機能を活用。3年間の学期レポートと進路情報から自動生成された下書きをベースに、具体的なエピソードを追加。通常1時間以上かかる作業が30分で完成しました。

4

新任の先生も質の高い進路指導

新任：山田先生

山田先生は初めての進路面談で不安を感じていましたが、生徒のアジェンダとレポートで事前準備ができました。面談中、生徒が「工学部に興味がある」と言った際も、RAG機能で「工学部 種類 違い」と検索し、その場で説明。先輩教員に頼らず質の高い指導ができました。

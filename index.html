学校・先生が安心して使える
進路支援ツール

株式会社PLUSは、学校・先生が安心して使える進路支援ツールを提供しています。

生徒の思いや考えを日常的に記録し、進路指導主事・学年主任の先生が
「探す時間」ではなく「向き合う時間」に集中できる環境をつくります。

VisionWell+

進路支援を変える、3つの視点

生徒にとって

  • ✓ AIに相談しながら考えを整理できる
  • ✓ 自分の気持ち・価値観を言葉にできる
  • ✓ 面談の場で「うまく話せない」を減らせる

教員にとって

  • ✓ 生徒の記録を一元管理
  • ✓ 調査票作成時に、過去の記録を探し回らなくていい
  • ✓ 面談前に生徒像を把握できる

学校にとって

  • ✓ 進路面談の正味時間を最大化
  • ✓ 指導の質が属人化しにくい
  • ✓ 記録が蓄積され、学年・年度を越えて活用できる
